# Dormitory Management System 

A desktop application built with WPF and SQL Server for managing dormitory operations such as student information, room allocation, and administrative management.

---

## Project Overview 

Dormitory Management System helps administrators manage:

- Student records
- Room allocation
- Dormitory information
- CRUD operations with SQL Server

This project demonstrates desktop application development using WPF with database integration.

---

## Technologies Used 

- C#
- WPF (.NET)
- SQL Server (LocalDB)
- ADO.NET
- Git & GitHub

---

## Database Setup

1. Open SQL Server Management Studio  
2. Connect to:  
   `(localdb)\MSSQLLocalDB`  
3. Open file:  
   `Database/Dormitory_Management.sql`  
4. Execute the script  
5. Run the application  

---

## Features 

- Add / Edit / Delete Students
- Display student list using DataGrid
- Page Navigation in WPF
- Database connection with SQL Server
- Basic input validation
